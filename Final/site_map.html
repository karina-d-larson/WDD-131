The Enchanted Bakery Site Plan

Karina Larson

WDD 131

Overview

Purpose

The purpose of this website is to present a magical woodland bakery where users can explore a collection of enchanted treats. The site allows users to browse items by category, search for specific treats, and interact with the content in a fun and engaging way.

Audience

The intended audience is people who enjoy fantasy and whimsical themes. The site is designed as a creative and immersive experience for users who like magical, story-inspired environments.

Dynamic elements

The website will use JavaScript to display and update bakery items dynamically. Items will be stored in an array of objects and displayed using functions. Users can filter items by category and search using input. The site will use DOM manipulation, event listeners, and conditional logic to respond to user interaction.

Content

Home page

The home page will include a welcome message, a short introduction, and a featured item. It will include an image and a visually engaging layout.

Menu Page

The menu page will display bakery items in a grid. Each item includes a name, image, description, and category. Users can filter and search items.

About Page

The about page will include a short story about the bakery’s magical woodland setting and theme, along with an image.
